An implementation of wrapper lameteoagricole.net to MQTT for home assistant. Creates a `weather.<location>` entity (compatible with the Météo France card) plus agri sensors (DJC, vigilance, UV, humidity, wind, sun/moon…) for one or several French locations.

Original Github : https://github.com/Smeagolworms4/meteoagricole2mqtt
